# Genora AI

> One prompt. Infinite possibilities.

Plataforma de criação com IA — imagens de alta qualidade powered by **Gemini (Nano Banana)**.

## Setup

```bash
npm install
cp .env.example .env.local
# Adicione sua KREA_API_KEY

npm run dev
```

Abra http://localhost:3000

## Variáveis de ambiente

```
KREA_API_KEY=sua_chave_aqui
```

Gere em: https://www.krea.ai/settings/api-tokens
